Theme Variants — Light & Dark Mode Components

Design Intent: Fenrir Ledger supports both light and dark themes. Dark mode is the default (matching existing Saga Ledger aesthetic). Light mode inverts backgrounds to warm cream/stone while preserving the gold accent color. Theme detection follows system preference (prefers-color-scheme) with manual toggle override.

Implementation Specifications

Element Dark Theme Light Theme Notes
Page Background --void (#12100e) --cream (#faf8f3) Warm tones in both modes
Card Background --forge (#1c1917) white (#ffffff) Slight elevation from page bg
Primary Text --text-saga (#f0ede4) --text-dark (#1c1917) High contrast for readability
Secondary Text --text-rune (#a09888) --text-dim (#6a6560) 60-70% opacity of primary
Gold Accent --gold (#d4a520) --gold (#d4a520) Unchanged - brand color
Borders --border (#3a3530) --border-light (#e0ddd4) Subtle definition
Theme Toggle Icon Moon (☾) Sun (☀) Shows current mode
Shadow/Glow Gold/ice-blue glows Subtle shadows only Less glow in light mode
Implementation Strategy:
1. Use CSS custom properties that switch based on [data-theme] attribute on body
2. Default to system preference via @media (prefers-color-scheme)
3. Store user's manual theme choice in localStorage
4. Theme toggle in nav updates data-theme and localStorage
5. All color values should reference CSS variables, never hardcoded hex
6. Test both themes for WCAG AA contrast compliance
7. Consider reduced motion preferences for theme transition animations
