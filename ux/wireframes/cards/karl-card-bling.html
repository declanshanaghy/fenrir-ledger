Karl Card Skin — Gold Borders, Runic Accents

Cosmetic card skin for Karl and trial tiers. Adds corner rune accents, gold-tinted border, hover glow animation, and gold-accented status badge. No flip animation — purely cosmetic overlay on the existing CardTile structure.

Scope: CardTile component only. CSS-only implementation via [data-tier="karl"] and [data-tier="trial"] cascade (data-tier set on <html> by EntitlementContext — already wired in #1086). New CSS class: .karl-bling-badge-status added to karl-bling.css. Rune corner spans added to CardTile.tsx.
