6. Interaction Spec

Trigger

No change to the avatar trigger button in the TopBar. Clicking/tapping the 28 px avatar button (44 px touch target) opens/closes the dropdown. aria-expanded toggles accordingly.

Profile Row Behaviour

  • The profile row is non-interactive — it is a visual identity label, not a button or link.
  • aria-hidden="true" on the row means screen readers skip it; user identity is already announced via the trigger button's aria-label.
  • No hover state, no focus state, no keyboard action.

Focus Management

No change from current behaviour:

  • When the dropdown opens, focus does not move to the profile row (it is aria-hidden).
  • The first focusable element inside the dropdown is "My Cards".
  • Escape closes the dropdown and returns focus to the avatar trigger.
  • Tab / Shift-Tab cycles through menu items only.

Long Text Edge Cases

  • Name, email, and tagline each truncate independently with text-overflow: ellipsis.
  • The avatar never shrinks (flex-shrink: 0) — text column absorbs all overflow.
  • At 375 px, the dropdown is 343 px wide (375 − 32 px margins). The text column has ~287 px available (343 − 8 gap − 40 avatar − 32 padding). This is sufficient for typical name/email lengths.
