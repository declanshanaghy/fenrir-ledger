Implementation Guidelines:

Typography (Both Themes):
• Headings: font-family: 'Cinzel', serif (matching About page)
• Body text: font-family: 'Source Serif 4', serif
• Code/technical: font-family: 'JetBrains Mono', monospace

Theme Detection:
• Use Next.js next-themes package (already in use on marketing site)
• CSS variables switch based on [data-theme="dark"] attribute
• Remove hardcoded dark background from chronicle.css

Color Variables to Update:
Light Theme:
• --background: #ffffff
• --foreground: #1a1a1a
• --muted: #6b6b6b
• --border: #e0e0e0
• --accent: #c9920a (gold stays similar)

Dark Theme:
• --background: #0f1018
• --foreground: #e8e4d4
• --muted: #8a8578
• --border: #1e2235
• --accent: #f0b429 (brighter gold)

Code Block Syntax Highlighting:
• Use theme-aware syntax highlighter (e.g., Prism with light/dark themes)
• Light theme: GitHub light color scheme
• Dark theme: GitHub dark color scheme

Accessibility:
• Maintain WCAG AA contrast ratios in both themes
• Light theme: 4.5:1 minimum for normal text
• Dark theme: Careful with muted text on dark backgrounds
• Test with browser dev tools contrast checker
