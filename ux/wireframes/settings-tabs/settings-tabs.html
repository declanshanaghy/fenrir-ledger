URL HASH PERSISTENCE: Active tab is stored in the URL hash (#account, #household, #settings). On page load, read location.hash and activate matching tab. Default: #account. On tab switch, update location.hash via history.replaceState (no page reload, no history entry).

MOBILE (<600px, shown at 375px)

MOBILE TAB PATTERN: Native <select> dropdown replaces the horizontal tab bar. Rationale: At 375px, 3 text tabs can feel cramped and may truncate. A native select is touch-friendly (44px min-height), accessible by default, and familiar on mobile. It also scales if more tabs are added later. The hamburger menu pattern was considered but rejected — it hides navigation behind a tap, reducing discoverability for only 3 items.

TABLET (600–1024px)

Tablet uses the DESKTOP tab bar (horizontal tabs still fit at 600px+). Card grid switches to single-column stacking (same as mobile) to avoid cramped 50/50 splits on narrow tablets. Cards get full width and stack. Breakpoint: md (768px) for grid switch — cards go 2-col only above 768px.

BLING TIER BEHAVIOR

data-tier="karl"

Tabs: Active tab gets gold border-bottom (2px solid #d4a520). Hover: gold glow animation (karl-gold-glow).

Cards: Existing karl-bling-card hover glow. Rune corners visible (0.55 opacity).

Mobile select: Wrapper gets gold border. Select text color remains default.

data-tier="trial"

Tabs: Active tab gets softer gold border-bottom (2px solid rgba(212,165,32,0.55)). Hover: soft gold glow (karl-gold-glow-soft).

Cards: Existing trial karl-bling-card treatment. Rune corners at 0.30 opacity.

Mobile select: Wrapper gets softer gold border.

data-tier="thrall"

Tabs: Plain border-bottom (default border color). No glow.

Cards: No bling classes fire. Standard border treatment.

Mobile select: Standard border, no gold.

NEW CSS CLASS needed: .karl-bling-tab — follows the same cascade pattern as .karl-bling-card and .karl-bling-btn in karl-bling.css. See interaction spec for the full CSS definition. Free trial users get Karl bling on tabs (same as cards/buttons — trial subset includes tabs).
