Error block: error type, timestamp, auto-retry countdown. "Retry Now" calls syncNow() immediately. "Dismiss Error" clears the error state visually but does not stop retries.

role="alert" on error block so screen readers announce it on first render.

G — Accessibility (WCAG 2.1 AA)

ARIA roles

  • Error block: role="alert" — announces on first render
  • Sync progress bar: role="progressbar" aria-label="Sync in progress"
  • "Sync Now" disabled state: aria-disabled="true" disabled
  • Status dot in heading: decorative — aria-hidden="true"
  • Upsell feature list: semantic <ul>
  • Settings card section: <section aria-label="Cloud Sync">

Focus & keyboard

  • All buttons keyboard-focusable with visible focus ring (focus-visible:ring-2)
  • Tab order: heading → timestamp → Sync Now button (natural DOM order)
  • Error block injected into DOM — focus does NOT auto-move to it (not a dialog)
  • Progress bar is non-interactive — skip in tab order (tabindex="-1")
