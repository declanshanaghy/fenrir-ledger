Panel 5 — Accessibility & Responsive Behavior

Concern Requirement
prefers-reduced-motion When OS/browser has reduced-motion enabled: skip wolf-rise, saga-shake, and ragnarok-pulse animations. FENRIR AWAKENS band still appears (no animation) for the 3s duration so the user is not excluded from the easter egg.
Screen readers status-band uses role="status" + aria-live="assertive". Wolf silhouette uses role="img" + aria-label. No interactive elements added during howl — keyboard focus is not moved.
Audio Optional. Only plays if AudioContext is already running (never autoplay). Must be accompanied by a user gesture previously in the session.
Mobile (< 640px) Wolf silhouette: full viewport width, height capped at 50vh so status band remains readable. FENRIR AWAKENS band: font-size scales down to 14px on mobile. saga-shake: reduced to ±3px on mobile to avoid layout jank on small displays.
Dismissal Howl is purely presentational — no dismiss button required. Sequence completes and all overlays fade automatically after ~3.8s total. ESC key does NOT dismiss (user cannot accidentally skip the experience).
Trigger listener cleanup useEffect cleanup removes the keydown listener on unmount. Sequence ref resets to [] on component unmount.
