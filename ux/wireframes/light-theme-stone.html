Typography Hierarchy

Headings are ink-black, NOT gold. Body text is dark charcoal/ink. Muted text is cool grey.

Heading — Ink Black (#14161a)
Body text — also ink black (#14161a). Clear and readable on marble white background. This is the primary text color for all content.
Muted/secondary text — cool dark grey (#5e6570). Used for metadata, timestamps, and auxiliary information. Links use burnt gold (#8a6a0a) for sufficient contrast.

Design Notes for Implementation

Key Decisions:

  • Cool stone/marble palette replacing warm parchment: All surfaces use 215deg-220deg hue (cool blue-grey) instead of 36deg-42deg (warm brown).
  • Ink-black headings, NOT gold: Headings use #14161a for maximum clarity. Gold reserved for accents and links.
  • Darker burnt gold for primary: #8a6a0a (38deg 75% 32%) instead of #ae8510 for better contrast on light backgrounds.
  • Subtle cool borders: Card borders are cool light grey (#b8bcc2), NOT warm tan. Blue undertone throughout.
  • Stone grain background texture: Radial gradient + fractal noise at 2% opacity (lighter than parchment theme's 3%).
  • Realm stone color adapted: --realm-stone changed from warm grey (37deg) to cool grey (215deg) to match stone aesthetic.

Component-Specific Notes:

  • StatusRing: All 4 realm states (Asgard, Hati, Muspel, Ragnarok) must be visible on light background. Stone state uses cool grey.
  • HowlPanel: Raven icon + urgent count must be visible on light background.
  • Easter egg modals: Remain dark overlay in both themes (design decision).
  • LCARS overlay: Remains dark overlay (Star Trek brand colors, unchanged).
  • Konami Howl: Remains dark full-screen takeover in both themes.
  • Loki Toast: Remains dark toast in both themes.

Responsive Behavior:

  • Mobile-first: All components must render correctly at 375px minimum viewport width.
  • Card grid: Auto-fill with minmax(280px, 1fr) — stacks to single column on narrow viewports.
  • Theme toggle: Accessible via TopBar dropdown on all screen sizes.

Accessibility:

  • WCAG AA contrast: All text/background pairs exceed 4.5:1 ratio (see table above).
  • Focus states: Burnt gold focus ring (#8a6a0a) with 1px + 12px shadow (same as dark theme, opacity adjusted).
  • Reduced motion: All animations respect prefers-reduced-motion (handled in globals.css).
