localStorage Gate — State Flow Diagram

Mermaid stateDiagram-v2 — follows ux/ux-assets/mermaid-style-guide.md conventions. 

```mermaid
stateDiagram-v2
    classDef primary fill:#03A9F4,stroke:#0288D1,color:#FFF
    classDef healthy fill:#4CAF50,stroke:#388E3C,color:#FFF
    classDef neutral fill:#9E9E9E,stroke:#757575,color:#FFF

    [*] --> CardSaved : User saves card

    CardSaved --> CountActive : Count active cards\n(status !== 'closed')
    CountActive --> CheckThresholds : Active count = N

    CheckThresholds --> GateCheck1 : N matches milestone?
    GateCheck1 --> AlreadySeen : localStorage key exists
    GateCheck1 --> FireToast : key absent
    AlreadySeen --> Done : suppress — no toast
    FireToast --> SetKey : Set egg:milestone-N
    SetKey --> ShowToast : Render toast (shadcn Toaster)
    ShowToast --> AutoDismiss : 5 seconds pass
    AutoDismiss --> FadeOut : opacity 1→0
    FadeOut --> Done : Remove from DOM

    Done : [*]
```

Re-fire prevention: once egg:milestone-N is set, the milestone NEVER fires again. Closing cards that drop count below threshold does NOT reset the gate. Milestones celebrate the add action, not the current portfolio size.
