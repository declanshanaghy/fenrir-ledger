Ragnarök Activation Rules
Urgent Card Count Mode UI Changes Browser Title Howl Header Konami Volume
0 Normal No change Fenrir Ledger The wolf is not howling. All chains are silent. 0.25
1–2 Warning (normal UI) HowlPanel appears; cards show urgent status rings Fenrir Ledger THE HOWL [N] 0.25
≥ 3 Ragnarök ACTIVE Red radial overlay + HowlPanel header override + title override Ragnarök — Fenrir Ledger Ragnarök approaches. Multiple chains tighten. 1.0

Statuses that count: fee_approaching, promo_expiring. Statuses that do NOT count: active, closed. Threshold is hard-coded in Sprint 4 — not configurable.

Konami Code Volume Escalation

During Ragnarök: Konami triggers howl at volume 1.0 (max intensity).

Outside Ragnarök: Konami triggers howl at volume 0.25.

KonamiHowl.tsx consumes RagnarokContext to read the flag and adjust audio gain node.

Accessibility Requirements (Non-Negotiable)

  • Overlay div carries aria-hidden="true" — invisible to screen readers
  • No WCAG AA contrast violations: red overlay at rgba(150,30,10,0.18) is below contrast threshold impact
  • All modals, dialogs, toasts remain fully interactable (pointer-events:none on overlay)
  • RagnarokContext change fires within one render cycle — no user action needed
  • Valhalla route excluded from overlay — closed cards are beyond the storm
