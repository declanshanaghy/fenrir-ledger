1 — Scope Overview

The ErrorBoundary.tsx fallback UI is a plain inline-styled div. This issue replaces it with a Norse-themed Loki Error Tablet — a compact inline component that fits within the monitor UI's existing panel layout.

Key distinction from NorseErrorTablet.tsx: that is a full-pane display for pod-level session errors (TTL expired, node unreachable). The Loki Error Boundary tablet is inline/compact — it wraps individual UI sub-components and should not take up the entire viewport. It adapts to the width of its container.

Personality: Loki — the trickster. He did not cause the error; he found it and bound it before it could cause more chaos. The tone is wry and confident, not apologetic. Loki is doing you a favour.

3 — Component States

State A — Error caught (primary)

Full tablet shown. Loki avatar, heading, subheading, error inscription, seal, retry button. All elements visible.

This is the hasError: true state in ErrorBoundary.

State B — Normal (children rendered)

Tablet hidden. ErrorBoundary renders this.props.children transparently. No DOM output from the tablet.

This is the hasError: false state. The Loki tablet has zero presence.

State C — Custom fallback prop

If fallback prop is provided to ErrorBoundary, it renders instead of the Loki tablet. The Loki tablet is the default fallback only.

Preserve existing if (this.props.fallback) return this.props.fallback guard.

State Transition: Retry

When the user clicks "Reweave the Thread":

  1. Button fires onClick
  2. setState({ hasError: false, error: null })
  3. React re-renders children
  4. If children throw again → State A returns immediately
  5. If children render successfully → Loki tablet disappears (State B)

No loading/pending state is needed between retry and result — the re-render is synchronous. A failed retry returns to State A instantly; no spinner required.

  • Avatar rune: 28px → 22px, letter-spacing: 8px → 5px
  • Heading: 16px → 14px
  • Subheading: 12px → 11px (may truncate to 2 lines — acceptable)
  • Rune border rows: overflow hidden + text-overflow ellipsis — partial row is fine (decorative)
  • Seal rune row: 14px → 12px, letter-spacing: 5px → 3px
  • Retry button: full-width on mobile (width: 100%) for easier tap target (min 44px height)
  • Body padding: 16px 20px → 12px 14px
  • Inscription code: 11px → 10px
