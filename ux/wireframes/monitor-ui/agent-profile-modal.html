Section 6 — State Machine: Open / Close

AgentProfileModal is controlled by a single state: profileAgent (string | null) held in the parent (App.tsx or LogViewer.tsx). null = closed.

State A — Closed
profileAgent = null
Modal not rendered (conditional render, not display:none).
Focus remains on last active element.
State B — Open
profileAgent = "firemandecko"
Modal rendered. Focus trapped inside. Backdrop visible.
Scroll body locked (overflow: hidden on <body>).
State C — Dismiss
Three triggers → profileAgent = null
1. Click ✕ button
2. ESC key (keydown handler on modal element)
3. Click backdrop (outside modal shell)
Focus returns to the trigger button that opened the modal (store ref before open).

Recommended: lift profileAgent state to App.tsx so any child (AgentBubble, JobCard, Sidebar) can trigger it via a callback prop or Context. Avoids prop-drilling through LogViewer. Alternative: React Context — AgentProfileContext.setProfileAgent(key).

Section 7 — Accessibility Spec (WCAG 2.1 AA)

RequirementImplementation
Modal role <div role="dialog" aria-modal="true" aria-labelledby="apm-name"> — "apm-name" is the agent name heading
Focus trap On open, focus moves to ✕ button (first focusable). Tab cycles: ✕ → (no other interactive elements) → ✕. Shift+Tab same cycle.
ESC key keydown listener on modal; Escape calls onClose(). Must work even if focus is on backdrop.
Focus return Store triggerRef before opening. On close, triggerRef.current?.focus().
Backdrop click onClick on backdrop div; check e.target === backdrop element (not modal shell) before closing.
Trigger buttons All three avatar wrappers are <button> elements (not divs). aria-label="View {name} profile".
Portrait alt text alt="{agentName}" — meaningful, not empty (image conveys identity).
Rune signature aria-label="Rune signature: {agentName}" on the rune <span> — screen reader reads latin name, not unpronounceable runes.
Rune bands (decorative) aria-hidden="true" on both rune band divs.
Body scroll lock document.body.style.overflow = 'hidden' on open; restored on close. Use useEffect cleanup.
Reduced motion If CSS entrance animation added — @media (prefers-reduced-motion: reduce) { animation: none }.
Color contrast Agent accent color must meet 4.5:1 against modal background for agent name. FiremanDecko (#4ecdc4), Loki (#a78bfa), Luna (#6b8afd) — verify against dark/light bg respectively.
