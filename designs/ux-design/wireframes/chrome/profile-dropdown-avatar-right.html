6. Interaction Spec

Trigger

The entire header identity cluster (name + email + avatar) is one <button>. Clicking/tapping it opens/closes the dropdown menu. aria-expanded toggles accordingly.

Dropdown Behaviour

  • Dropdown contains only menu items — no profile/identity information.
  • First focusable element is "My Cards".
  • Escape closes and returns focus to trigger.
  • Click outside closes the dropdown.

Responsive Behaviour

  • Desktop (sm+): Name + email visible in header, right-aligned text left of avatar.
  • Mobile (<sm): Name + email hidden (hidden sm:flex). Avatar-only trigger. Dropdown unchanged.

Keyboard Navigation

KeyBehaviour
Enter / Space on triggerOpen dropdown, focus first menu item (My Cards)
TabMove to next menu item
Shift+TabMove to previous menu item
EscapeClose dropdown, return focus to trigger
Click outsideClose dropdown
