Accessibility Requirements
Element Requirement
Modal container role="dialog". aria-modal="true". aria-labelledby bound to H1 id. Focus trap: Tab cycles within dialog.
H1 heading Receives focus on dialog open. Acts as the dialog's accessible name via aria-labelledby.
Data summary block role="region". aria-label="Anonymous data found". Screen readers announce before the choice buttons.
"Import N cards" button Dynamic label including count. Min 44px touch target. Receives initial focus on dialog open.
"Start fresh" button Same visual weight as "Import" button. Min 44px touch target. Must not be styled to imply "danger" — it is a valid choice.
Atmospheric copy Eyebrow: aria-hidden="true". Footer quote: readable but not a focusable element. Both are Voice 2 — decorative framing, not action copy.
