Card Wizard Step Transitions — Animation Specification

Animation Direction: Steps slide based on navigation direction. Forward (Next/More Details) slides left, revealing new content from right. Backward (Back) slides right, revealing previous content from left. This creates a natural left-to-right reading progression.

Framer Motion Implementation

// AnimatePresence wrapper for step transitions <AnimatePresence mode="wait" initial={false}> {currentStep === 1 && ( <motion.div key="step1" custom={direction} initial="enter" animate="center" exit="exit" variants={{ enter: (direction: number) => ({ x: direction > 0 ? '100%' : '-100%', opacity: 0 }), center: { x: 0, opacity: 1 }, exit: (direction: number) => ({ x: direction < 0 ? '30%' : '-30%', opacity: 0 }) }} transition={{ x: { duration: 0.25, ease: [0.16, 1, 0.3, 1] }, opacity: { duration: 0.2 } }} > {/* Step 1 Content */} </motion.div> )} </AnimatePresence>

Step Indicator Dots Animation

// Step dot component with animated state <motion.div className="step-dot" animate={{ scale: isActive ? [1, 0.9, 1] : 1, backgroundColor: isActive ? '#gold' : '#muted' }} transition={{ scale: { duration: 0.3, ease: [0.16, 1, 0.3, 1] }, backgroundColor: { duration: 0.2, delay: 0.1 } }} aria-label={`Step ${step} of 2: ${stepTitle}`} />

Accessibility: prefers-reduced-motion

// CSS media query for reduced motion @media (prefers-reduced-motion: reduce) { .step-panel { transition: none !important; animation: none !important; } // In React/Framer Motion const prefersReducedMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches; transition={{ duration: prefersReducedMotion ? 0 : 0.25, ease: [0.16, 1, 0.3, 1] }} }

Implementation Notes:
• Use Framer Motion's AnimatePresence with mode="wait" to prevent layout jumps
• Track direction state: 1 for forward, -1 for backward navigation
• Exit animations are shorter (200ms) than enter (250ms) for perceived speed
• 50ms overlap between exit and enter creates fluid motion
• Step dots animate 100ms after slide begins to avoid competing for attention
• All animations respect prefers-reduced-motion with instant swaps
