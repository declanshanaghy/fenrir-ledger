Step Indicator — Accessibility & Animation Specification

Purpose: The step indicator provides visual and semantic feedback about the user's position in the multi-step card creation wizard. It must be accessible to keyboard and screen reader users while providing smooth visual transitions.

Basic Step Indicator (Dots)

Default State — Step 1 Active
HTML Structure:
• Wrapped in <nav role="tablist"> for semantic navigation
• Each dot is a <button role="tab"> for keyboard interaction
• aria-selected indicates current step
• aria-label provides full context: "Step X of Y: Description"
• tabindex manages focus (0 for current, -1 for others)

Step Indicator with Progress Line

Enhanced with Visual Connection
Progress Line: Visual connection between steps. Animates color/width when advancing. Position: absolute, z-index: -1 to sit behind dots.

React Implementation with Framer Motion

// StepIndicator.tsx import { motion } from 'framer-motion'; interface StepIndicatorProps { currentStep: number; totalSteps: number; onStepClick?: (step: number) => void; } function StepIndicator({ currentStep, totalSteps, onStepClick }: StepIndicatorProps) { const stepTitles = [ "Card and Bonus Details", "Additional Information" ]; return ( <nav role="tablist" aria-label="Card creation progress"> <div className="flex justify-center items-center gap-4 py-4"> {Array.from({ length: totalSteps }, (_, i) => { const stepNumber = i + 1; const isActive = stepNumber === currentStep; const isPast = stepNumber < currentStep; return ( <button key={stepNumber} role="tab" aria-selected={isActive} aria-label={`Step ${stepNumber} of ${totalSteps}: ${stepTitles[i]}`} tabIndex={isActive ? 0 : -1} onClick={() => onStepClick?.(stepNumber)} className="relative w-11 h-11 flex items-center justify-center cursor-pointer focus:outline-none group" > <motion.div className={`w-2 h-2 rounded-full border-2 ${isActive ? 'bg-gold border-gold' : isPast ? 'bg-muted border-muted' : 'bg-white border-muted'}`} animate={{ scale: isActive ? [1, 0.9, 1.2, 1] : 1, borderWidth: isActive ? '2px' : '2px' }} transition={{ scale: { duration: 0.3, ease: [0.16, 1, 0.3, 1], times: [0, 0.3, 0.6, 1] } }} whileHover={{ scale: isActive ? 1 : 1.1 }} whileFocus={{ boxShadow: '0 0 0 3px rgba(201, 146, 10, 0.3)' }} /> {/* Screen reader only text */} <span className="sr-only"> {isActive && "(Current) "} Step {stepNumber} of {totalSteps}: {stepTitles[i]} </span> </button> ); })} </div> </nav> ); }

Keyboard Navigation Behavior

Key Action Implementation
Tab Move focus to step indicator Only current step has tabindex="0"
Arrow Left/Right Navigate between steps (optional) Change focus and optionally trigger step change
Enter/Space Activate focused step (if clickable) Call onStepClick handler
Escape Exit step navigation Move focus to next focusable element

ARIA Attributes

Attribute Element Purpose Example Value
role="tablist" Container nav Identifies step indicator as navigation tabs —
role="tab" Each step button Identifies individual step —
aria-selected Each step button Indicates current step "true" or "false"
aria-label Container nav Describes the navigation purpose "Card creation progress"
aria-label Each step button Full step description "Step 1 of 2: Card and Bonus Details"
tabindex Each step button Controls keyboard focus order 0 (current) or -1 (others)
