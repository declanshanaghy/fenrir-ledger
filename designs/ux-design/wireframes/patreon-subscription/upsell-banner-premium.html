Component Spec: UpsellBanner

Reusable component for all 8 premium feature upsells. Rendered by a PremiumFeatureGate wrapper that checks auth + tier status. 

Component: UpsellBanner

Props:
  feature: "cloud-sync" | "multi-household" | "advanced-analytics"
           | "priority-import" | "data-export" | "extended-history"
           | "custom-notifications" | "cosmetic-perks"

Render condition:
  - User is signed in (not anonymous)
  - User is NOT Karl tier (tier !== "karl")
  - Maximum 1 UpsellBanner per page (PremiumFeatureGate manages this)

Behavior:
  - "Learn more" click: opens HardGateModal for the given feature
  - Not individually dismissible
  - Disappears when user upgrades to Karl

Accessibility:
  - role="complementary"
  - aria-label="Premium feature: {Feature Name}"
  - "Learn more" link: keyboard focusable, aria-label="Learn more about {Feature Name}"
  - min-height 44px on "Learn more" link for touch target

Responsive:
  - Desktop: horizontal layout (rune left, content right)
  - Mobile (<640px): vertical stack (rune above content)
